How to Use the MemMachine Playground

To get started with the MemMachine Playground, follow these simple steps.

  1. Click the "Go to Playground" button.

    Go to Playground

  2. On the login page, click "Continue with Google" to create a secure account and log in. The email and password feature is not enabled.

  3. In the model selection dropdown at the top of the page, click the '+' to add both of these models to create a side-by-side comparison:

    • gpt-4.1-mini: This model has no memory and will not remember information in a new chat session.
    • gpt-4.1-mini-memory: This model uses the power of MemMachine to remember facts and information within and across chat sessions.

  4. Once both models are added, you can start chatting. You can compare the response from both models simultaneously.

  5. Start a new chat session by clicking the "New Chat" button on the left sidebar. Continue chatting. You will notice the gpt-4.1-mini forgets everything from the previous session(s), whereas the gpt-4.1-mini-memory retains context and information from all prior sessions.
