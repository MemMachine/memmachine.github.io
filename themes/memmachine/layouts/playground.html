How to Use the MemMachine Playground

To get started with the MemMachine Playground, follow these simple steps.

  1. Click the "Go to Playground" button.

    Go to Playground

  2. On the left, enter your Hugging Face API Token/Key to login. Your token must have READ access.

  3. Select a model from the dropdown

  4. Select "Compare with MemMachine"

  5. Continue chatting and start new sessions. You will notice the Control model forgets everything from the previous session(s), whereas the MemMachine model retains context and information from all prior sessions.
